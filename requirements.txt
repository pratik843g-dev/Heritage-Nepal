# Heritage Nepal - Quick Setup for New Users
# ============================================
# JUST PULLED THE PROJECT? Follow these steps:
# ============================================

# STEP 1: Make sure you have Node.js installed
#   Download from: https://nodejs.org
#   Required: Node.js v18.0.0 or higher
#   Check version: node --version

# STEP 2: Install all dependencies (run in terminal)
#   Command: npm install
#   Wait: 2-3 minutes
#   This downloads all packages automatically

# STEP 3: Run the project
#   Command: npm run dev
#   Open browser: http://localhost:5173
